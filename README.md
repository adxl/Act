# Act

### Run the project

- [OPTIONAL] Install dev-dependencies: `$ [npm|yarn] install`
- Run server container: `$ docker compose up -d`
- Navigate to `localhost:5000`

### Features:

- [x] native objects prototypes
- [x] prop_access
- [x] String interpolate
- [x] objects creation
- [x] class inheritance
- [x] modules
- [x] routing
- [x] promises
